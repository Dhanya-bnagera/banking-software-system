# Banking Software System (Demo)

Simple demo banking app using Flask + SQLite for SDLC assignment.

Features:
- Register / Login
- View Balance
- Deposit / Withdraw
- Transfer between users
- Transaction history

Run:
1. pip install -r requirements.txt
2. python app.py
3. Open http://127.0.0.1:5000
Use /setup-demo to create demo users (alice/bob).
